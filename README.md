# Learn Next

This repository is dedicated to my learning journey with the **Next.js** framework, exploring its native features and focusing on a clean and efficient architecture.

## 🚀 About the Project

The goal of this project is to master Next.js as a full-stack framework based on React, making the most of the optimizations it offers without the need for additional packages whenever possible.

### Key Features Under Study:

- **Routing:** New file-system based routing (`App Router`).
- **API Routing:** Building integrated *BFFs* (Backends for Frontends).
- **Rendering:** Different rendering strategies (SSR, SSG, ISR).
- **Styling:** Recommended styling approaches by the framework.
- **Optimization:** Automatic performance improvements in build and compilation.

## 📚 Documentation

The technical documentation for this learning process is being built using **Writerside** (JetBrains) and is automatically published via GitHub Actions.

You can find the detailed topics in `Writerside/topics/`, including:
- `Default-topic.md`: Overview and why learn Next.js.
- `Creating-a-new-App.md`: Guide to initializing the project.

## 🛠️ Technologies

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Writerside](https://www.jetbrains.com/writerside/)

## ⚙️ CI/CD

The project includes a GitHub Actions workflow (`.github/workflows/workflow.yml`) that:
1. Builds the documentation using Writerside.
2. Runs automatic documentation checks.
3. Deploys to **GitHub Pages**.
