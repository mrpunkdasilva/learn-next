# Learn NextJS

## What's this?

Framework fullstack baseado no React, focado em ser uma otimização de algumas features, focar em ser coersivo em arquitetura e algumas features que são melhoradas são:
- routing: possui um novo tipo de routing que é pensado para ser basado em pastas
- build & compiling: possui melhorias para o build e como é focado em usar com TypeScript possui otimizações na compilação
- e mais

> A ideia não é instalar packages adicionais e sim usar todo o que o Next possui pois ele já forneceria "tudo"

## Why learn Next?

Porque é muito rapido de construir web apps e já deixar production-ready, por causa de algumas features:

1. Routing
2. API Routing: é possivel construir _BFF_, que são os backends for frontend, send o APIs mais simples que estariam dentro do frontend assim conseguiriamos ter construido um app fullstack de maneira mais rápida
3. Rendering
4. Styling
5. Optimization
